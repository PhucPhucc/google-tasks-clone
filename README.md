# google-tasks-clone

```
  git clone https://github.com/PhucPhucc/google-tasks-clone.git
  cd ./google-tasks-clone
```

# run backend
```
  cd ./backend
  npm run dev
```
# run frontend
```
  cd ./frontend
  npm run dev
```
